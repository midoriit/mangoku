# 萬石以上分限帳
『吹塵録』「萬石以上分限帳」による、幕末における一万石以上の藩とその石高をマップにしました。

GitHub Pagesによる表示は [こちら](https://midoriit.github.io/mangoku/)

データはLinkData.orgの[萬石以上分限帳](http://linkdata.org/work/rdf1s5180i)でCC-BYライセンスで公開しています。
